# Smart-Plug-Testing

Olen junior QA / seadmete testimise huviline, kellel on kogemus koduautomaatika seadmete funktsionaalsuse testimisel, vigade tuvastamisel ja dokumenteerimisel.

Projekt demonstreerib nutipistiku käsitsi testimist reaalsetes kasutustingimustes. 
Projekti eesmärk oli hinnata seadme töökindlust, ühenduvust ja käitumist erinevates olukordades.

## Eesmärk

Projekti eesmärk on näidata minu oskusi:
- seadmete testimisel
- vigade tuvastamisel
- testimise dokumenteerimisel
- probleemide analüüsimisel

# Oskused

- Seadmete funktsionaalsuse testimine
- Manual testing
- Bug reportide koostamine
- Testjuhtumite loomine
- Vigade tuvastamine ja analüüs
- Troubleshooting

## Õppisin projekti käigus

- Kuidas dokumenteerida testimistulemusi
- Kuidas koostada bug report'e
- Kuidas analüüsida seadmete käitumist erinevates olukordades
- Kuidas teha korduvtestimist pärast muudatusi
- Kui oluline on täpne dokumentatsioon testimisel

## Testimiskeskkond

- Seade: Smart Plug
- Ühendus: WiFi
- Testimise tüüp: Manual testing
- Platvorm: Android mobiilirakendus
- Testimise keskkond: Kodune võrk
