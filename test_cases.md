### Test 1: Esmane seadistus

## Eesmärk: kas seade ühendub edukalt WiFi-ga

# Sammud:

- Lülita seade sisse
- Ava äpp
- Lisa seade
- Ühenda WiFi

# Oodatav tulemus: seade ühendub edukalt

### Test 2: Võrgu kadumine

## Eesmärk: kuidas seade käitub WiFi katkedes

# Sammud:

- Lülita seade tööle
- Lülita ruuter välja
- Oota 1–2 minutit

# Oodatav tulemus: seade kaotab ühenduse ja taastub hiljem

### Test 3: Elektrikatkestus

## Eesmärk: kas seade taastub pärast voolu kadumist

# Sammud:

- Lülita seade sisse
- Tõmba vool pistikust
- Pane tagasi

# Oodatav tulemus: seade käivitub uuesti

### Test 4: Mitu telefoni

## Eesmärk: kas mitu kasutajat saavad seadet juhtida

# Sammud:

- Ühenda seade telefon A-ga
- Lisa telefon B
- Juhi seadet mõlemast

# Oodatav tulemus: mõlemad saavad juhtida

### Test 5: Äpi restart

## Eesmärk: kas juhtimine töötab pärast äpi sulgemist

# Sammud:

- Juhi seadet
- Sulge äpp
- Ava uuesti
- Proovi uuesti juhtida

# Oodatav tulemus: töötab normaalselt
